# Issue drafts

This folder contains draft issue descriptions for features identified during a repository survey of club/activities management systems.

Files are named with a numeric prefix and can be used with the provided script to open GitHub issues via the `gh` CLI.

To publish all drafts as GitHub issues run:

```bash
./scripts/create_issues.sh
```

You must be authenticated with the GitHub CLI (`gh auth login`) and have permission to create issues in this repository.
